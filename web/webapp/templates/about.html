General

This website holds the data captured at monitoring stations. This is essentially a measure of biodiversity.

Currently, there is only one monitoring station. This is located near Clifton Hampden, Oxfordshire. (See Position)

Wikipedia

Some of the data used in the species pages comes from the excellent resource that is Wikipedia and Wikidata. where this is used is stated on each page, but in general, the descriptions and photos of species comes from there. Thank you to the people who make there work free so that websites like this one can exist.

Data

There are several types of data, Analyses, Detections, Detectors, Positions and Species; each is described below.

Analyses

These are the processes that have run on the data to identify a species. This might be a simple piece of software, an AI trained identifier, or even a human.

See a list of analyses here.

Detections

These store a detection of a single species, when it happened and how it was found. Only detections with a higher confidence than 0.500 are included in this website.

See a list of detections here.

Detectors

These are the physical devices used to capture a piece of data. A camera or microphone for example.

See a list of detectors here.

Positions

This is the location of the monitoring station. (i.e. where all the detectors are located)

See a list of positions here. (Only one currently!)

Species

These are all the species detected. A Gray Wagtail or a Long-tailed Tit.

See a list of species here.
